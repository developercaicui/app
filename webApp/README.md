

## 编译设置

```
# 安装依赖
npm install

# 启动服务
npm start

# 打包压缩
npm start
```

## 技术栈 vue  + vue-router + vuex + webpack + ES6/7 + axios + sass + flex

## 项目结构

```
.
├── build                                         // 配置文件
├── config                                        // 环境配置区分
├── dist                                          // 项目文件
├── srcIpad                                       // Ipad源码
├── srcMobile                                     // 手机源码
│   ├── mock                                      // 假数据        
│   ├── api                                       // 公用Api接口封装
│   ├── port                                      // 线上接口
│   ├── components                                // 木偶组件
│   ├── containers                                // 智能组件
│   ├── base                                      // 基础组件（如下拉刷新）
│   ├── router                                    // 路由
│   ├── store                                     // 状态机
│   └── assets                                    // 资源
│
│   ├── App.vue                                
│   ├── main.js                                
│
.

```
