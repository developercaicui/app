

## 编译设置

```
# 安装依赖
npm install

# 启动服务
npm start

# 打包压缩
npm start
```

## 技术栈 vue  + vue-router + vuex + webpack + ES6/7 + axios + sass + flex

## 项目结构

```
.
├── build                                       // 配置文件
├── config                                      // 环境配置区分
├── dist                                        // 项目文件
├── src                                         // 源码目录
│   ├── mock                                    // 数据交互假数据
│
│   ├── api                                     // 常用接口封装
│   ├── components                              // 组件
│   ├── Ipad                                    // Ipad页面组件
│   ├── Mobile                                  // 手机页面组件
│   ├── router
│   │   ├── ipad.js                             // ipad路由配置
│   │   ├── mobile.js                           // 手机路由配置
│   │   └── index.js                            // 路由入口
│ 
│   ├── store                                   // vuex状态管理
│
│   └── assets                                  // 资源文件
│
│   ├── App.vue                                 // 页面入口文件
│   ├── main.js                                 // 程序入口文件
│
├── index.html                                  // 入口html文件
.

```
